# 飞书收藏插件

一个浏览器插件，帮助您将网页收藏保存到飞书多维表格中，支持标签、备注和自动同步功能。

## 功能特性

- 🚀 **一键收藏**：点击浏览器工具栏图标即可收藏当前网页
- 🏷️ **标签管理**：为收藏添加自定义标签，方便分类管理
- 📝 **备注功能**：添加详细的备注信息，记录收藏原因
- 🔄 **自动同步**：自动同步到飞书多维表格，支持离线缓存
- 🔍 **智能搜索**：支持按标题、URL、标签、备注搜索收藏
- 📊 **数据管理**：本地数据管理，支持导入导出
- ⚙️ **灵活配置**：自定义同步频率、存储上限等设置

## 安装使用

### 1. 开发环境安装

1. 克隆或下载本项目
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"，选择项目文件夹
5. 插件安装完成，会在工具栏显示图标

### 2. 飞书配置

1. 登录 [飞书开放平台](https://open.feishu.cn/)
2. 创建企业自建应用，获取 App ID 和 App Secret
3. 创建多维表格，获取 App Token 和 Table ID
4. 在插件设置页面填写相关信息

### 3. 使用插件

1. 浏览任意网页，点击工具栏的插件图标
2. 在弹窗中编辑标签和备注信息
3. 点击"收藏到飞书"按钮
4. 收藏会自动保存到本地并同步到飞书多维表格

## 飞书开放平台配置

### 获取应用凭证

1. 进入飞书开放平台，创建企业自建应用
2. 在"凭证与基础信息"中获取：
   - 应用 ID (App ID)
   - 应用密钥 (App Secret)

### 获取多维表格信息

1. 创建或打开一个飞书多维表格
2. 获取 App Token：从表格URL中提取，格式如 `bascnxxxxx`
3. 获取 Table ID：右键点击数据表名称，选择"获取ID"

### 权限配置

确保应用具有以下权限：
- `bitable:app` - 多维表格权限
- `bitable:record` - 记录管理权限

## 字段配置

插件会自动在多维表格中创建以下字段：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 标题 | 文本 | 网页标题 |
| 网址 | 文本 | 网页URL |
| 描述 | 文本 | 页面描述 |
| 标签 | 文本 | 收藏标签，逗号分隔 |
| 备注 | 文本 | 用户备注 |
| 图标 | 文本 | 网站图标URL |
| 关键词 | 文本 | 页面关键词，逗号分隔 |
| 收藏时间 | 文本 | 收藏时间戳 |
| 来源 | 文本 | 固定为"浏览器插件" |

### 主要功能模块

- **Popup界面**：提供收藏编辑界面
- **Content脚本**：获取页面信息
- **Background脚本**：处理核心业务逻辑
- **FeishuAPI**：封装飞书API调用
- **StorageManager**：管理本地存储
- **Options页面**：插件设置界面

### API接口

插件使用飞书开放平台的以下API：

- **认证**：`/auth/v3/tenant_access_token/internal`
- **获取表格元数据**：`/bitable/v1/apps/{app_token}/tables/{table_id}/fields`
- **添加记录**：`/bitable/v1/apps/{app_token}/tables/{table_id}/records`
- **批量添加**：`/bitable/v1/apps/{app_token}/tables/{table_id}/records/batch_create`
- **查询记录**：`/bitable/v1/apps/{app_token}/tables/{table_id}/records`
- **更新记录**：`/bitable/v1/apps/{app_token}/tables/{table_id}/records/{record_id}`
- **删除记录**：`/bitable/v1/apps/{app_token}/tables/{table_id}/records/{record_id}`

## 数据同步

### 同步策略

- **自动同步**：每30分钟自动同步未同步的记录
- **手动同步**：在设置页面点击"立即同步"按钮
- **实时同步**：收藏时如果网络正常，立即同步到飞书

### 离线处理

- 网络断开时，收藏记录保存在本地
- 网络恢复后，自动同步未同步的记录
- 支持本地搜索和管理，无需网络连接

## 隐私说明

- 插件仅在用户主动点击收藏时获取网页信息
- 所有数据都保存在用户的浏览器本地和飞书账户中
- 不会收集或上传用户的个人信息到第三方服务器
- 不会跟踪用户的浏览行为
