# 飞书收藏插件安装指南

## 📦 快速安装

### 方法1：开发者模式安装（推荐）

1. **下载插件代码**
   ```bash
   git clone https://github.com/your-username/feishu-collection-plugin.git
   # 或下载ZIP文件并解压
   ```

2. **打开Chrome扩展管理页面**
   - 在Chrome地址栏输入：`chrome://extensions/`
   - 或点击右上角菜单 → 更多工具 → 扩展程序

3. **开启开发者模式**
   - 在右上角开启"开发者模式"开关

4. **加载插件**
   - 点击"加载已解压的扩展程序"按钮
   - 选择插件文件夹（feishu-collection-plugin）
   - 插件图标会出现在浏览器工具栏

### 方法2：打包安装

1. **打包插件**
   ```bash
   # 进入插件目录
   cd feishu-collection-plugin
   
   # 打包成zip文件
   zip -r feishu-collection-plugin.zip . -x "*.git*" "node_modules/*"
   ```

2. **加载打包文件**
   - 打开 `chrome://extensions/`
   - 开启开发者模式
   - 拖拽zip文件到扩展页面
   - 或点击"加载已解压的扩展程序"选择解压后的文件夹

## ⚙️ 首次配置

### 1. 飞书开放平台配置

#### 创建应用
1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 登录后点击"创建应用"
3. 选择"企业自建应用"
4. 填写应用信息：
   - 应用名称：飞书收藏插件
   - 应用描述：浏览器收藏同步工具

#### 获取应用凭证
1. 进入应用管理页面
2. 点击"凭证与基础信息"
3. 记录以下信息：
   - 应用ID (App ID)
   - 应用密钥 (App Secret)

#### 配置权限
1. 点击"权限管理"
2. 添加以下权限：
   - `bitable:app` - 多维表格管理
   - `bitable:record` - 记录管理
   - `tenant:tenant:readonly` - 租户信息读取

### 2. 多维表格配置

#### 创建多维表格
1. 打开飞书，创建新的多维表格
2. 命名为"网页收藏"或其他你喜欢的名称

#### 获取表格信息
1. **获取App Token**：
   - 打开多维表格
   - 查看URL，格式：`https://xxx.feishu.cn/base/bascnxxxxx`
   - 复制`bascnxxxxx`部分作为App Token

2. **获取Table ID**：
   - 在多维表格中，右键点击数据表名称
   - 选择"获取ID"
   - 复制Table ID

#### 设置表格字段
插件会自动创建以下字段，你也可以手动创建：

| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| 标题 | 文本 | 网页标题 |
| 网址 | 文本 | 网页URL |
| 描述 | 文本 | 页面描述 |
| 标签 | 文本 | 收藏标签 |
| 备注 | 文本 | 用户备注 |
| 图标 | 文本 | 网站图标 |
| 关键词 | 文本 | 页面关键词 |
| 收藏时间 | 文本 | 收藏时间戳 |
| 来源 | 文本 | 固定为"浏览器插件" |

### 3. 插件配置

#### 打开设置页面
1. 点击浏览器工具栏的插件图标
2. 点击右上角的设置按钮
3. 或右键插件图标 → 选项

#### 填写配置信息
在设置页面填写：
- **应用ID**：从飞书开放平台获取
- **应用密钥**：从飞书开放平台获取
- **多维表格Token**：从多维表格URL获取
- **数据表ID**：从多维表格获取

#### 测试连接
1. 点击"测试连接"按钮
2. 如果配置正确，会显示"连接测试成功"
3. 如果失败，请检查配置信息

## 🎯 使用插件

### 收藏网页
1. 浏览任意网页
2. 点击浏览器工具栏的插件图标
3. 在弹窗中：
   - 查看页面信息
   - 添加标签（可选）
   - 添加备注（可选）
4. 点击"收藏到飞书"

### 管理收藏
1. 打开插件设置页面
2. 切换到"收藏管理"标签
3. 查看所有收藏记录
4. 搜索、筛选、删除收藏

### 同步数据
- **自动同步**：每30分钟自动同步
- **手动同步**：点击"立即同步"按钮
- **实时同步**：收藏时立即同步（网络正常）

## 🔧 常见问题

### Q: 插件安装失败？
**A:** 请检查：
- Chrome版本是否支持Manifest V3
- 是否开启了开发者模式
- 文件夹路径是否正确

### Q: 连接测试失败？
**A:** 请检查：
- 应用ID和密钥是否正确
- 应用是否已发布（需要发布才能使用）
- 网络连接是否正常
- 权限是否配置完整

### Q: 收藏同步失败？
**A:** 请检查：
- 多维表格Token和Table ID是否正确
- 表格字段是否匹配
- 网络连接是否稳定
- 查看控制台错误信息

### Q: 数据丢失怎么办？
**A:** 
- 本地数据：插件会自动备份，可在设置中导出
- 飞书数据：飞书有多版本历史，可恢复
- 建议定期导出数据备份

### Q: 如何卸载插件？
**A:**
1. 打开 `chrome://extensions/`
2. 找到飞书收藏插件
3. 点击"移除"按钮
4. 确认卸载

   - 查看package-info.json

4. **提交问题**
   - 在GitHub Issues提交
   - 提供详细的错误信息和复现步骤
