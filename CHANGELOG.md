## [1.0.0] - 2025-01-10

### ✨ 新增功能
- 🚀 **基础收藏功能** - 一键收藏当前网页到飞书多维表格
- 🏷️ **标签管理** - 支持自定义标签，方便分类管理
- 📝 **备注功能** - 添加详细的备注信息
- 🔄 **自动同步** - 每30分钟自动同步到飞书多维表格
- 🔍 **智能搜索** - 支持按标题、URL、标签、备注搜索
- 📊 **数据管理** - 本地数据管理，支持导入导出
- ⚙️ **设置管理** - 完整的插件配置界面
- 🎨 **现代化UI** - 响应式设计，支持深色模式
- 📱 **移动端适配** - 在手机浏览器上也能正常使用
- 🛡️ **安全保护** - 数据加密存储，保护用户隐私

### 🔧 技术特性
- 采用Chrome Extension Manifest V3
- 使用飞书开放平台API
- 支持离线缓存和断点续传
- 模块化代码结构
- 完整的错误处理机制
- 性能优化和内存管理

### 📋 支持的字段
- 标题、网址、描述、标签、备注
- 图标、关键词、收藏时间、来源
- 自动同步状态和时间戳

### 🎯 兼容性
- Chrome 88+ (支持Manifest V3)
- Edge 88+ (Chromium内核)
- 其他Chromium内核浏览器

### 📦 包含文件
- manifest.json - 插件配置
- background.js - 后台脚本
- content.js - 内容脚本
- popup/ - 弹窗界面
- options/ - 设置页面
- lib/ - 工具库
- icons/ - 图标资源
- utils.js - 工具函数
- test.html - 测试页面

## 版本历史说明

### 版本号规则
- **主版本号(MAJOR)**：不兼容的API修改
- **次版本号(MINOR)**：向下兼容的功能性新增
- **修订号(PATCH)**：向下兼容的问题修正

### 标签说明
- ✨ **新增功能** (Added) - 新增加的功能
- 🔧 **修复问题** (Fixed) - 修复的bug
- 🔄 **功能变更** (Changed) - 功能变更
- ⚡ **性能优化** (Performance) - 性能提升
- 🔒 **安全更新** (Security) - 安全相关更新
- 📚 **文档更新** (Documentation) - 文档变更

## 参与开发

欢迎提交Issue和Pull Request来改进这个插件！

### 提交格式
```
类型: 简短描述

详细说明...

### 新增
- 功能1
- 功能2

### 修复
- 问题1
- 问题2

### 变更
- 变更1
- 变更2
```

### 类型标签
- `feat`: 新功能
- `fix`: 修复bug
- `docs`: 文档更新
- `style`: 代码格式
- `refactor`: 代码重构
- `test`: 测试相关
- `chore`: 构建/工具
